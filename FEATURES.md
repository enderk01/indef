# ✨ Recursos - Pet ProjectX

## 🤖 IA Local

- **Modelo:** Qwen2.5-1.5B quantizado
- **Sem Internet:** Funciona completamente offline
- **Privado:** Suas conversas não são enviadas a ninguém
- **Rápido:** ~50-100ms de latência

### Capacidades de Conversa

```
Usuário: "Oi, tudo bem?"
Pet: "Oi! Estou feliz em conversar com você! 😊"

Usuário: "Me conte uma piada"
Pet: "Por que o livro de matemática se suicidou? Porque tinha muitos problemas! 😄"

Usuário: "Estou triste"
Pet: "Que pena... posso fazer algo para melhorar seu dia?"
```

## 🎮 Movimento Consciente

### Estados Emocionais

| Emoção | Comportamento | Cor |
|--------|--------------|-----|
| 😊 Feliz | Pula, dança | Ouro |
| 😢 Triste | Fica parado | Azul |
| 🤔 Curioso | Explora | Roxo |
| 😲 Excitado | Pula muito | Vermelho |
| 😴 Dormindo | Imóvel | Azul claro |

### Sistema de Energia

- **Energia:** Diminui com o tempo, restaura com descanso
- **Felicidade:** Aumenta com interações positivas
- **Auto-regulação:** Pet dorme quando cansado

## 💬 Interações

### Clique no Pet
- Responde com animação
- Aumenta felicidade
- Gasta energia

### Digite Mensagens
- Pet detecta emoção
- Responde contextualmente
- Muda comportamento accordingly

### Histórico de Chat
- Últimas 3 mensagens visíveis
- Contexto mantido durante conversa
- Pode limpar histórico

## 🎨 Interface

- **Minimalista:** Sem distrações
- **Responsiva:** Roda em baixos specs
- **Customizável:** Editar cores, tamanho, etc.

## 🚀 Performance

| Ação | Latência |
|------|----------|
| IA gerando resposta | 50-200ms |
| Renderização | 16ms (60 FPS) |
| Clique do pet | Imediato |

## 📊 Estatísticas em Tempo Real

- Energia (0-100%)
- Felicidade (0-100%)
- Estado emocional atual
- Tempo desde última interação
